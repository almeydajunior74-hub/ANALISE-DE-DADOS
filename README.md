# ANALISE-DE-DADOS

Projeto de análise de dados utilizando Python, Pandas e Plotly para identificar os principais fatores que levam ao cancelamento de clientes.

Tecnologias
Python
Pandas
Plotly
Jupyter Notebook
Objetivo

Analisar uma base de dados com 50.000 clientes e identificar padrões de comportamento relacionados ao cancelamento de serviços.

Etapas da Análise
Importação e tratamento dos dados
Remoção de valores nulos
Análise estatística dos cancelamentos
Criação de gráficos interativos
Identificação dos principais motivos de cancelamento
Principais Resultados
Taxa inicial de cancelamento: 56%
Clientes com contrato mensal apresentaram maior índice de cancelamento
Clientes com mais de 4 ligações ao call center tendem a cancelar o serviço
Clientes com mais de 20 dias de atraso apresentam alta probabilidade de cancelamento

Após a aplicação dos filtros identificados na análise, a taxa de cancelamento foi reduzida para aproximadamente 18%.
